Claude Code terminalinde çalışan agentic bir kodlama aracıdır. Kurulumu birkaç dakika sürer.

Claude Code is an agentic coding tool that runs in your terminal. Setup takes just a few minutes.

  • Node.js 18+ — check with node -v

  • macOS, Linux, or Windows (WSL recommended)

04 — Context

Context Yönetimi

Context Management

Context penceresi sınırlıdır. Bunu yönetmek performansın anahtarıdır.

The context window is limited. Managing it is key to performance.

🧹

/clear

Tamamen sıfırla. Her yeni görevde kullan.

Full reset. Use with every new task.

📦

/compact

Akıllı sıkıştırma. Neyi koruyacağını belirt.

Smart compression. Specify what to keep.

📊

/context

Doluluk oranını görselleştir.

Visualize fill rate.

🔀

/branch

Dallandır — alternatif dene.

Fork — explore alternatives.

⚠️ Context Rot

Uzun konuşmalarda Claude erken mesajları unutur. Görev değişiminde /compact veya /clear kullan.

In long conversations Claude forgets earlier messages. Use /compact or /clear when switching tasks.

GörevTask→ ÇalışWork→ /compact→ Yeni görevNew task→ Uzun? /clearLong? /clear

CLAUDE.md
# Project: E-Commerce API

## Commands
- Build: `npm run build`
- Test: `npm test -- --watch`
- Lint: `npm run lint:fix`

## Architecture
- Node.js + Express + TypeScript
- PostgreSQL + Prisma ORM
- JWT auth + refresh token rotation

## Rules
- All new code = TypeScript
- Functional components + hooks only
- JSDoc on every public function
- Throw errors, never return null

## Watch Out
- DO NOT touch /api/v1/legacy/*
- Run migrations: `npx prisma migrate dev`
- Never put secrets in .env

07 — Otomasyon07 — Automation

Hooks

Araç kullanımı öncesinde/sonrasında otomatik çalışan komutlar.

Commands that auto-run before/after tool use. Automate linting, formatting.

.claude/settings.json
{
  "hooks": {
    "PostToolUse": [
      {
        "matcher": "Write(*.py)",
        "hooks": [{ "type": "command", "command": "python -m black $file" }]
      },
      {
        "matcher": "Write(*.ts)",
        "hooks": [{ "type": "command", "command": "npx prettier --write $file" }]
      }
    ]
  }
}

PreToolUse

Araçtan önce. Doğrulama, güvenlik.

Runs before tool. Validation, security.

PostToolUse

Araçtan sonra. Format, lint, type check.

Runs after tool. Format, lint, type check.

Stop

Claude durduğunda. Final kontrol.

When Claude stops. Final verification.

💡 Hooks vs CLAUDE.md

Hook'lar deterministik — her seferinde çalışır. CLAUDE.md talimatları atlanabilir. Linter/formatter → hook.

Hooks are deterministic — they run every time. CLAUDE.md instructions can be skipped. Linter/formatter → use hooks.
