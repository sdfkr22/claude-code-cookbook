## 📖 What is this?

**Claude Code Cookbook** is a curated collection of practical patterns and prompts for using Claude effectively in coding workflows.

This is **not a project showcase**.

It is a **cookbook** — a place to quickly find:
- Useful prompt patterns
- Clear and minimal explanations
